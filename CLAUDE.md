## Markdown Formatting Rules (load before writing, not after)

Both imported files above are mandatory context for this repository.
Before creating a new `.md` file or editing an existing one, confirm
`FORMATTING_RULES.md` has been read and its "Before Starting" section
applied while drafting. Do not treat formatting compliance as a pass to
run only after the document is written; that checklist is only the final
verification, not the first read.
